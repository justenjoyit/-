# 网络协议分层

| 5层协议 | 7层协议    | 作用                         | 协议                     | 其他                  |
| ------- | ---------- | ---------------------------- | ------------------------ | --------------------- |
| 应用层  | 应用层     | 为应用程序提供服务           | HTTP/HTTPS/FTP/SMTP/POP3 |                       |
| 应用层  | 表示层     | 数据格式转换，数据加密       |                          |                       |
| 应用层  | 会话层     | 建立，管理，维护会话         |                          |                       |
| 传输层  | 传输层     | 建立，管理和维护端到端的连接 | TCP/UDP                  | 差错控制/流量控制     |
| 网络层  | 网络层     | IP选址和路由选择             | IP                       | 路由器                |
|         | 数据链路层 | 提供介质访问和链路管理       |                          | MAC/LLC子层，差错检测 |
|         | 物理层     | 物理                         |                          |                       |
